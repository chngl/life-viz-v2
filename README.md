# life-viz-v2

How to run the app (assuming you already have node.js on your machine)

clone the repo:
```
  git clone https://github.com/chngl/life-viz-v2.git
```
install gulp and httpster:
```
  npm install --global gulp
  npm install --global httpster
```
go to the app root path and build it:
```
  cd /path/to/life_viz_v2/
  gulp build
```
go to the dist folder and start the app:
```
  cd /path/to/life_viz_v2/dist/
  httpster
```  
That's it. Now hit 127.0.0.1:3333 and check the viz :)
